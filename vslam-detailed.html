三、图像预处理
3.1 去畸变 (Undistortion)
  • 透镜材质导致径向畸变和切向畸变
  • 畸变模型：
    x_distorted = x(1 + k1*r² + k2*r⁴) + 2p1*x*y + p2*(r²+2*x²)
  • 通过相机标定获取内参矩阵和畸变系数
  • 标定常用棋盘格法
3.2 图像滤波 (Filtering)
  • 高斯滤波：去除高斯噪声，平滑图像
  • 中值滤波：去除椒盐噪声，保护边缘
  • 双边滤波：保边去噪，但计算量大
  • 目的：提高特征提取的鲁棒性
3.3 相机模型
  • 针孔模型：小孔成像，投影关系
  • 内参矩阵 K = [fx 0 cx; 0 fy cy; 0 0 1]
  • 投影：s * [u,v,1]ᵀ = K * [X,Y,Z,1]ᵀ
  • 外参：旋转R + 平移t
四、前端：视觉里程计 (Visual Odometry)
4.1 特征点提取
  • ORB (Oriented FAST + BRIEF)：最快，综合性能好
  • SIFT：精度最高，但专利收费
  • SURF：SIFT加速版
  • FAST：只检测角点，速度极快
  • 提取流程：FAST找角点 → Harris筛选 → 计算ORB描述子
4.3 运动估计方法
  • 2D-2D：对极几何
    五点法/七点法求解基础矩阵F，再分解得到E和R,t
  • 3D-2D：PnP
    已知3D点坐标和2D投影，求相机位姿
  • 3D-3D：ICP
    两组3D点配准，最小化距离误差
  • 常用算法：P3P, EPNP, DLT, RANSAC
4.2 特征匹配
  • 暴力匹配 (BF)：计算所有描述子距离，排序取最近
  • 汉明距离 (Hamming)：用于ORB的二进制描述子
  • 快速最近邻 (FLANN)：高维特征更快
  • 互一致性检测：A的最近邻是B，且B的最近邻是A
  • 对极约束筛选：
    x₂ᵀ * F * x₁ = 0 剔除outlier
  • RANSAC：随机采样一致性剔除误匹配
4.4 三角化 (Triangulation)
s₁x₁ = P₁X
s₂x₂ = P₂X
用两帧的投影矩阵P₁,P₂和图像坐标x₁,x₂，解算3D点X
  • 深度恢复：单目无法直接获知深度，需多帧三角化
  • 基线(baseline)越大，深度精度越高
五、后端：图优化 (Backend Optimization)
5.1 为什么要后端？
  • 问题：累积误差
  • VO每帧估计都有误差
  • 误差随时间逐帧累积
  • 结果：轨迹漂移(drift)，地图裂开
  • 100帧后误差可达1-10米
5.2 位姿图 (Pose Graph)
  • 节点：相机位姿 (R,t)
  • 边：帧间约束（运动方程）
  • 每条边对应一个误差项
  • 图优化 = 调整节点位置，使整体误差最小
5.3 BA (Bundle Adjustment)
min Σ || z - h(T, X) ||²
调整所有位姿和路标点，使重投影误差最小
  • 重投影误差：3D点投影到图像与实际观测的差异
  • 本质：大规模非线性最小二乘
  • 求解：高斯牛顿法 / LM(Levenberg-Marquardt)
六、回环检测 (Loop Closure)
6.1 问题
  • 机器人走了一圈回到起点
  • 理论：起点终点的位姿应该相同
  • 现实：由于累积误差，对不上
  • 结果：地图出现"裂痕"
6.2 解决方法
  • 检测：当前帧 vs 历史帧
    相似度 > 阈值 → 发现回环
  • 词袋模型 (BoW)：
    将图像转为"词袋"向量，快速比对
  • DBoW2/3：常用库，将ORB描述子转为词袋
  • 几何验证：用RANSAC验证一致性
  • 添加回环边 → 触发全局优化

八、核心挑战与解决方案
☀️ 挑战1：光照变化
  • 强光/逆光 → 特征点丢失
  • 阴影/光斑 → 误匹配
  • 傍晚/夜间 → 基本不可用
💡 解决方案
  • 多传感器融合 (+IMU/+LiDAR)
  • HDR高动态范围成像
  • 学习式特征 (CNN-based)
🏃 挑战2：动态物体
  • 行人/宠物/车辆移动
  • 被误认为特征点 → 匹配错误
  • 导致轨迹估计偏差
💡 解决方案
  • 动态物体检测 (SegNet/MaskRCNN)
  • 剔除移动特征点
  • 只保留静态特征点建图
💻 挑战3：计算资源
  • 特征提取：ORB约15ms/帧
  • BA优化：可长达秒级
  • 嵌入式设备算力有限
💡 解决方案
  • 多线程并行处理
  • 关键帧(KeyFrame)策略
  • 边缘化(Marginalization)减少变量

十、一句话总结
VSLAM = 眼睛(摄像头) + 记忆(特征点) + 大脑(后端优化)
提取特征 → 匹配特征 → 估计运动 → 优化轨迹 → 检测回环 → 生成地图
